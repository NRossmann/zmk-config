## Hardware Features

### Split Design
- **Left Hand:** Letters, numbers, symbols, modifiers
- **Right Hand:** Numbers, navigation, editing
- **Thumb Clusters:** Spacing, layers, modifiers

### Key Switches
- Mechanical switches (Cherry MX compatible)
- German QWERTZ layout
- Ergonomic split design

### RGB LEDs
- Per-key RGB lighting
- Underglow LEDs
- Customizable colors, brightness, animations

### Bluetooth
- Up to 5 paired devices
- USB fallback option
- Easy device switching

---

## Tips & Tricks

### Efficient Typing
1. **Symbols without reaching:** Use Layer 1 for all special characters
2. **Editing combos:** F-keys + navigation on Layer 2
3. **One-handed:** Use modifiers with opposite hand

### Layer Switching
- **Hold = temporary:** Layer buttons work while held
- **Double-tap = switch:** (if configured) Can toggle layers
- **Priority:** LY2 (nav) and LY3 (RGB) can be held simultaneously

### Customization Ideas
- Add mod-tap for faster typing
- Create custom macros for common phrases
- Configure combos for faster symbol entry
- Adjust RGB animations to match your style
